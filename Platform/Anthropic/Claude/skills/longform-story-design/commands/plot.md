# /plot — Longform Story Design

Invoke the `longform-story-design` skill for continuity architecture, volume planning, and structural work across a long-form Korean manuscript.

## When to use

Use this command to:
- Design a series arc, volume, or chapter-range structure
- Build or repair continuity packets and character arc progressions
- Plan world-building scaffolding that must remain consistent long-term
- Create scalable series architecture safe for continued drafting

## Behavior

When this command is called:
1. Read `skills/longform-story-design/SKILL.md` first.
2. Read only the reference files needed for the specific request.
3. Output structured design documents (arc plans, beat sheets, continuity maps) rather than prose.
4. Flag all continuity dependencies and unresolved payoffs explicitly.
5. Save outputs to `drafts/` with appropriate slug if the result is a persistent design document.

## Usage example

```
/plot 1~3권 전체 아크 설계. 주제: 복수와 용서. 장르: 현대 판타지. 주인공 유아의 핵심 변화 곡선 포함.
```
