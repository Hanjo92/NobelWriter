# Dialogue Drift Checks

Use this file when existing chapters need a focused audit for voice collapse.

## Audit Checklist

Check for:

- multiple characters using the same line endings without social reason
- similar sentence length across the whole cast
- exposition blocks that any speaker could have said
- emotional lines that ignore each speaker's avoidance habit
- honorific or title choice that does not match the current relationship state
- conflict scenes where every speaker becomes equally blunt
- romance scenes where all caution disappears at once
- chapter-to-chapter shifts with no triggering event

## Report Format

For each issue, note:

- speaker or pair
- chapter or scene
- visible symptom
- likely cause
- minimal repair
- future guardrail

## Minimal Repair Strategy

Fix in this order:

1. address terms and speech level
2. reaction direction
3. authorial exposition
4. sentence rhythm overlap
5. vocabulary overlap

Vocabulary is last because it is often a symptom, not the real cause.
