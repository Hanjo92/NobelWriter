# Dialogue Drift Lexicon

## 1. Exposition Disguised As Dialogue

### Bad

```text
민준: 네가 지난번 프로젝트에서 팀을 배신하고 경쟁사로 자료를 넘긴 일 때문에 아직도 다들 널 못 믿는 거야.
서윤: 그때는 어쩔 수 없었고, 우리 팀이 얼마나 힘들었는지 너도 알잖아.
```

### Fixed

```text
민준: 그래서 아직도 널 못 믿는 거야.
서윤: 너도 이유는 알잖아.
```

## 2. Both Speakers Know It Already

### Bad

```text
지후: 우리 아버지가 삼 년 전에 돌아가신 뒤로 네가 우리 집을 계속 도와줬잖아.
나림: 그러니까 내가 지금 널 그냥 둘 수 없는 거야.
```

### Fixed

```text
지후: 이제 와서 왜 또 끼어드는데.
나림: 그냥 두는 쪽이 더 이상해서.
```

## 3. Symmetrical Anger

### Bad

```text
도현: 나도 화났어.
가람: 나도 화났거든.
도현: 그래서 더 실망이야.
가람: 나도 실망이야.
```

### Fixed

```text
도현: 화난 쪽은 나야.
가람: 그래, 넌 늘 그 말부터 하지.
도현: 웃기지 마.
가람: 웃긴 건 지금도 네가 피해자인 척하는 거고.
```

## 4. Premature Register Collapse

### Bad

```text
주임: 팀장, 그건 아니잖아요.
팀장: 야, 넌 빠져.
주임: 나도 할 말은 해야죠.
```

### Fixed

```text
주임: 팀장님, 그 판단은 다시 보셔야 합니다.
팀장: 선 넘지 마.
주임: 넘는 거 압니다. 그래도 말씀드려야 합니다.
```

## 5. Generic Comfort

### Bad

```text
소라: 괜찮아질 거야.
민재: 다 잘될 거야.
소라: 너무 걱정하지 마.
```

### Fixed

```text
소라: 아직 끝난 거 아니야.
민재: 그런 말이 무슨 소용인데.
소라: 소용 있게 만들 거야. 물부터 마셔.
```

## 6. Rival Voices Collapsing Into One Tone

### Bad

```text
민준: 왜 말 안 했어.
서윤: 너도 말 안 했잖아.
민준: 그래서 더 화나.
서윤: 나도 화나.
```

### Fixed

```text
민준: 왜 숨겼어. 내가 모를 줄 알았냐.
서윤: 말 안 한 건 맞아. 근데 네가 들을 상태도 아니었어.
민준: 판단은 내가 해.
서윤: 늘 그렇게 말하지. 그래서 내가 입 다문 거고.
```

## 7. Apology Formula With No Cost

### Bad

```text
유나: 미안하다고 했잖아.
태경: 나도 미안해.
유나: 그럼 됐지 뭐.
```

### Fixed

```text
유나: 미안하단 말로 끝낼 거야?
태경: 그럴 생각 아니었어. 진짜 미안해.
유나: 뭐가 미안한데.
태경: 네가 혼자 버티게 만든 거. 그거부터 잘못했어.
```

## 8. Pre-Romance Turning Explicit Too Fast

### Bad

```text
서하: 사실 나 너 좋아해.
지운: 나도 너 좋아해.
서하: 그럼 우리 사귀자.
```

### Fixed

```text
서하: 오늘 연락 없더라.
지운: 늦을 줄 몰랐어.
서하: 몰랐으면 지금은 알겠네.
지운: ...다음엔 먼저 말할게.
```

## 9. Family Dialogue Too Clean

### Bad

```text
누나: 네 건강이 걱정돼서 그래.
동생: 알겠어. 네 마음 이해해.
누나: 나도 네 입장을 이해해.
```

### Fixed

```text
누나: 약은 먹었어.
동생: 또 그 얘기야?
누나: 또 안 먹었으니까 그렇지.
동생: ...먹을게. 됐지.
```

## 10. Office Disagreement Too Blunt

### Bad

```text
수빈: 그 결정은 틀렸어요.
태준: 네가 뭘 알아요.
수빈: 저는 맞고 당신은 틀렸어요.
```

### Fixed

```text
수빈: 이 상태로 넘기면 검수에서 걸려요.
태준: 오늘 안 넘기면 일정이 밀립니다.
수빈: 밀리는 것보다 터지는 게 더 커요.
태준: ...좋아요. 대신 수정 범위는 줄입시다.
```

## 11. Threat By Volume Only

### Bad

```text
도식: 내가 진짜 죽여버릴 거야.
진우: 나도 안 무서워.
도식: 진짜라고.
```

### Fixed

```text
도식: 지금 그 문 열면 네 이름부터 올라간다.
진우: 겁주면 멈출 줄 알아?
도식: 겁주는 게 아니야. 순서 알려주는 거지.
```

## 12. Evasion Without A Visible Core

### Bad

```text
세린: 그 파일 네가 옮겼어?
우진: 글쎄, 세상일이라는 게 다 그런 거지.
세린: 무슨 말이야?
우진: 그냥 여러 가지가 있지.
```

### Fixed

```text
세린: 그 파일 네가 옮겼어?
우진: 지금 그게 중요해?
세린: 중요하니까 묻지.
우진: 옮기긴 했어. 근데 이유는 아직 말 못 해.
```

## 13. Everyone Using The Same Filler

### Bad

```text
하린: 아니, 그러니까 그게 아니고.
이준: 아니, 그러니까 내 말은.
하린: 아니, 그러니까 넌 항상.
```

### Fixed

```text
하린: 그렇게 경계만 하면 재미없지 않아?
이준: 재미 보러 온 건 아닌데.
하린: 그럼 왜 자꾸 내 쪽을 봐.
```

## 14. Pauses Used As Decoration

### Bad

```text
가람: ...그러니까...
도현: ...나는...
가람: ...그게...
```

### Fixed

```text
가람: ...그 얘긴 지금 하지 마.
도현: 왜. 찔려?
가람: 아니. 그냥, 지금은 아니라고.
```

## 15. Group Scene Round-Robin

### Bad

```text
팀장: 의견 말해봐.
민호: 저는 걱정됩니다.
주희: 저도 걱정됩니다.
도윤: 저도 같은 생각입니다.
```

### Fixed

```text
팀장: 지금부터는 보고 없이 움직이지 않습니다.
민호: 그럼 이미 놓친 건 어떡합니까.
주희: 따지기 전에 인원부터 다시 세죠.
도윤: 세면 뭐 합니까. 안에 남은 사람이 있는데.
팀장: 그래서 더 질서가 필요하다는 겁니다.
```
