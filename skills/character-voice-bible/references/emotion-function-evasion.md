# Emotion Function: Evasion

Use this file when a speaker must avoid direct answer without sounding random.

## Core Rule

Hide through structure, not fog.

Useful evasion moves:

- answer a smaller adjacent question
- redirect to timing or procedure
- return a question
- admit one part to hide the core part
- stall with attitude, humor, or offense

## Common Shape

```text
A: pressure
B: substitute answer
A: push for the missing core
B: second dodge or partial reveal
```

## Sample

```text
세린: 그 파일 네가 옮겼어?
우진: 지금 그게 중요해?
세린: 중요하니까 묻지.
우진: 옮기긴 했어. 근데 이유는 아직 말 못 해.
```

## What To Check

- is the avoided core still visible to the reader
- does the dodge match the speaker's personality
- does evasion buy time, shift blame, or redirect pressure clearly
