# Emotion Function: Interrogation

Use this file when one speaker must corner the other into a narrower answer.

## Core Rule

Pressure should tighten with each turn.

Good interrogation dialogue:

- narrows the target
- cuts off escape routes
- turns vague answers into liabilities
- makes silence or delay meaningful

## Common Shape

```text
A: question or accusation
B: partial answer or dodge
A: narrower question or proof
B: reluctant admission, counter, or deeper dodge
```

## Sample

```text
가람: 너 어제 어디 있었어.
도현: 왜 그걸 지금 묻는데.
가람: 대답부터 해. 어디 있었냐고.
도현: ...병원.
```

## What To Check

- does the pressure get narrower, not merely louder
- does the target speaker lose room over time
- does each line force a more specific position
