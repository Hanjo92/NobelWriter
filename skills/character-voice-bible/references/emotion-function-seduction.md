# Emotion Function: Seduction

Use this file when a speaker should attract, tempt, or lower the other's guard through dialogue.

## Core Rule

Do not make seduction equal blunt confession.

Useful moves:

- strategic softness
- playful pressure
- selective attention
- inviting ambiguity
- making refusal sound like continued engagement

## Common Shape

```text
A: low-risk lure
B: caution or dry block
A: closer lure with more personal focus
B: partial yield, sharper caution, or curious return
```

## Sample

```text
하린: 그렇게 경계만 하면 재미없지 않아?
이준: 재미 보러 온 건 아닌데.
하린: 그럼 왜 자꾸 내 쪽을 봐.
이준: ...안 본다고는 안 했어.
```

## What To Check

- does attraction move through tempo and attention rather than explicit labeling
- does one speaker invite while the other manages distance
- does the exchange stay asymmetrical
