# Emotion Function: Soothing

Use this file when one speaker must calm, steady, or de-escalate the other.

## Core Rule

Soothing should lower temperature without erasing tension.

Useful moves:

- reduce pace
- name the next immediate action
- refuse panic without dismissing it
- replace abstract fear with one controllable step

## Common Shape

```text
A: panic, anger, or spiral
B: calmer anchor
A: resistance or overflow
B: firmer grounding or practical next step
```

## Sample

```text
소라: 지금 끝난 거야. 다 망했어.
민재: 아직 아니야. 숨부터 고르자.
소라: 그런 말이 무슨 소용인데.
민재: 소용 있게 만들 거야. 물부터 마셔.
```

## What To Check

- does the soothing speaker remain distinct rather than generic
- does calm come with method, not empty reassurance
- does the distressed speaker resist in character
