# Micro Voice: Pauses And Interruptions

Use this file when silence, hesitation, false starts, or interruption style should carry voice.

## Core Rule

Pauses must mean something.

Useful pause functions:

- buying time
- swallowing emotion
- recalibrating register
- baiting the other speaker to fill silence
- showing loss of control

## Sample Contrast

```text
A: ...그 얘긴 지금 하지 마.
B: 왜. 찔려?
A: 아니. 그냥, 지금은 아니라고.
```

## What To Check

- does the pause delay the answer for a reason
- does one speaker cut in while another waits
- are ellipses overused as decoration
- do false starts fit the speaker's stress pattern

## Drift Warning

Too many pauses flatten everyone into the same hesitant voice.
