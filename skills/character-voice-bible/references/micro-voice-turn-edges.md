# Micro Voice: Turn Openings And Closings

Use this file when speakers need recognizable ways to enter and leave a turn.

## Core Rule

Openings and closings should support reaction logic.

Opening tendencies:

- start bluntly
- start with a softener
- start by negating the prior line
- start by reframing procedure

Closing tendencies:

- end with a command
- end with a cut-off
- end with a baiting question
- end with reluctant concession
- end with unresolved residue

## Sample Contrast

```text
A: 됐고, 본론만 말해.
B: 아니요, 그 전에 하나만 확인할게요.
```

## What To Check

- who enters the turn by attacking versus cushioning
- who leaves a clean stop versus an open wound
- whether openings and closings repeat with controlled consistency

## Drift Warning

Do not force every line to begin or end with a signature marker. The pattern should be noticeable only over several turns.
