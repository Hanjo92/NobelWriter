# Reaction Patterns

Use this file when character differentiation should come from how a person reacts under pressure.

## Pressure Axes

Separate speakers by combining:

- direct vs indirect
- compressed vs expansive
- confrontational vs evasive
- self-exposing vs self-concealing
- soothing vs escalating

## Example Reaction Lenses

### Anger

- cut short and cold
- sarcastic and needling
- over-explaining and defensive
- politely sharper, not louder
- silent withdrawal before one precise strike

### Fear

- denial
- practical problem-solving
- joke deflection
- status compliance
- attachment-seeking

### Care

- direct reassurance
- disguised nagging
- action instead of verbal comfort
- distance plus logistical help
- awkward partial confession

## Repair Rule

If two characters use the same words but should still feel different, change:

1. how fast they answer
2. how much they expose
3. whether they attack, shield, dodge, or stall
4. whether they end the turn cleanly or leave emotional residue
