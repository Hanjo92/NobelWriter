# Pre-Romance Dialogue

Use this file when attraction exists but neither side can say it cleanly yet.

## Core Rule

Let feeling leak through restraint, not direct declaration.

Useful moves:

- complaint that is really care
- joke that masks exposure
- refusal to answer too quickly
- partial concession instead of confession

## Common Shape

```text
A: low-risk probe or complaint
B: dodge or dry answer
A: second probe with more exposure
B: partial opening that still preserves face
```

## Sample

```text
서하: 오늘 연락 없더라.
지운: 늦을 줄 몰랐어.
서하: 몰랐으면 지금은 알겠네.
지운: ...다음엔 먼저 말할게.
```

## What To Check

- does intimacy move without full collapse of restraint
- do both speakers avoid becoming equally candid at once
- does the exchange leave residue instead of clean closure
