# Script Dialogue Rules

Use this file when the output should read like a script made mostly of spoken turns.

## Core Rule

Each line must do something to the prior line.

Acceptable functions:

- answer
- evade
- challenge
- soften
- escalate
- stall
- bait
- confess partially
- deny
- redirect

If a line does none of these, cut or rewrite it.

## Default Format

Prefer:

```text
Character A: line
Character B: line
```

Add brief action beats only when the exchange becomes unclear without them.

## What To Avoid

- long setup lines that summarize the whole scene
- duplicated information across adjacent turns
- two speakers using the same rhythm for three or more turns in a row
- lines that sound like narration converted into quotation marks
- over-explaining emotional meaning instead of letting pressure show it

## Turn Tests

For every turn, ask:

1. What pressure arrived from the previous line?
2. Does this speaker meet it, dodge it, or turn it?
3. Would another character answer in the same structure?
4. Did the emotional temperature change, even slightly?

## Minimal Action Beat Rule

Only insert a beat when it changes the speech:

- silence before answering
- forced laugh
- cutting in
- looking away instead of answering
- physically blocking exit

Do not add decorative gestures that do not alter the next line.
