# Voice Diagnostic Questions

Use this file when the user gives too little information to lock a voice cleanly.

## Ask Only The Questions That Unblock Voice Design

- Who can this character speak plainly to, if anyone?
- Who makes this character become more polite, colder, or more evasive?
- What does this character avoid saying directly even when it matters?
- Under anger or embarrassment, do they cut short, over-explain, joke, dodge, or go silent?
- What is the fastest way this character reveals care?
- What relationship shift is supposed to show up in the dialogue now?

## If You Cannot Ask

When follow-up is impractical, infer from manuscript evidence in this order:

1. status handling
2. address forms
3. pressure behavior
4. emotional exposure
5. rhythm and directness

Keep the assumption narrow. State it once, then continue.
