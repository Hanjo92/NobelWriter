# Omnibus Structure

Use this file when the user wants an omnibus-style longform, rotating-focus series, or loosely episodic structure.

## Core Rule

`옴니버스` is a structure mode, not a genre package.

Keep it in `Format` or structure notes. Still choose the dominant genre package separately.

## What Counts As Omnibus

Treat the project as omnibus-structured when one or more of these are true:

- each arc or episode can center a different lead or focal pair
- the setting or institution is more stable than the POV cast
- the series promise comes from variation inside a shared frame
- closure happens repeatedly at episode or arc scale, not only at the final ending

## What It Does Not Decide

`옴니버스` does not decide whether the package is:

- `romance / romance fantasy`
- `progression / action fantasy`
- `mystery / thriller`
- `political / war / faction drama`
- `survival / apocalypse / horror`

Choose the package by return engine exactly as usual.

## Common Omnibus Shapes

- anchored omnibus: one stable institution, district, guild, school, or court with rotating focal characters
- braided omnibus: several recurring cast lines that alternate and converge
- case omnibus: semi-closed episode arcs tied by one hidden rule, space, or operator
- relationship omnibus: each arc spotlights a different pair inside one social order

## Intake Rule

If the user says `옴니버스`, record it like this:

- `Format`: omnibus web novel, omnibus long novel, rotating-focus serial, or equivalent
- `Genre package`: choose separately
- `Package selection reason`: explain the package, not the omnibus label

## Design Rule

For omnibus longform, lock these early:

- what stays stable across arcs
- what rotates across arcs
- what cumulative continuity must carry over
- what the reader returns for beyond simple novelty
- how each arc closes while still widening the whole

## Failure Risks

Watch for these first:

- arc variety without cumulative pressure
- new lead rotation that resets emotional investment too often
- world repetition because only focal characters change
- anthology feel without longform carryover

## Short Example

```text
Format: omnibus fantasy web novel
Genre package: mystery thriller
Package selection reason: The reader returns for how each case reveals the hidden rule of the city, so mystery thriller governs the structure even though the cast focus rotates.
```
