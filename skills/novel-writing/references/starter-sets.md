# Chapter Starter Sets

Use this file as the entry point when the user asks for a chapter 1 start, first-scene launch, or a ready-made opening package.

## How To Use

1. Pick the main genre.
2. Open the matching starter file.
3. Pick the closest sub-type or launch mode.
4. Use the checklist and draft block for `초고`.
5. Use the revision block for `개고`.
6. Apply the `Audience Tendency Modifiers` and `Quick Variant Blocks` inside the chosen genre file.

Keep the start narrow. A starter set should launch pressure fast, not explain the whole novel.

## Selection Rule

Choose by the main promise of chapter 1:

- `traditional fantasy`: oath, kingdom, relic, succession, expedition
- `martial arts`: sect pressure, rivalry, disgrace, duel, jianghu reputation
- `modern fantasy`: anomaly in ordinary life, awakening, hidden rules, guild or work pressure
- `fusion fantasy`: regression advantage, mixed systems, empire leverage, strategic imbalance
- `romance`: attraction, distance change, reunion, forced proximity, emotional misread
- `horror`: dread, violated safety, return, pursuit, contamination
- `mystery`: contradiction, clue, missing truth, testimony mismatch, investigation pressure
