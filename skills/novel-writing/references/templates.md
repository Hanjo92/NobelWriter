# Novel Writing Templates

Use these templates when the user wants structured planning or revision output instead of freeform advice.

For dialogue-only revision or character voice tuning, use the designated dialogue or voice skill instead of the non-dialogue revision templates in this file.

## Gaego Brief

Use this when the user wants `개고`, `퇴고`, or a cleaner second-pass rewrite rather than a brand-new draft.

```text
Title or working label:
Current source stage: [초고 / user excerpt / mixed]
Requested return stage: [개고 / 원고]
Genre and audience:
Scope: [scene / chapter / excerpt]
Keep fixed:
- core event:
- POV:
- factual meaning:
- character relationship status:

Improve first:
- narration:
- exposition / information delivery:
- pacing:
- continuity:
- tone/register:

Do not do:
- add new plot beats unless needed for clarity
- flatten the scene into summary
- over-polish into a different style target

Expected result:
- what should feel cleaner after the pass:
```

## Gaego Pass Card

Use this when you need a compact execution card before revising prose into `개고`.

```text
Pass goal:
Source text problem:
1. ...
2. ...
3. ...

Preserve:
- scene intent:
- emotional direction:
- key reveals:

Revise:
- remove translation-like syntax
- fix lines that explain shared context in narration
- sharpen subject/image landing in non-dialogue prose
- cut explanatory drag in setting or connective sentences
- keep world or procedural information readable on first pass

Exit check:
- reads naturally on first pass
- still feels like the same scene
- clearly cleaner than the source text
```

## Expository Paragraph Revision Card

Use this for setting explanation, world rules, procedural explanation, or connective prose that is not centered on a character's spoken voice.

```text
Paragraph role: [setting / world rule / transition / action explanation / recap bridge]
Primary problem: [too dense / too abstract / too slow / repetitive / translation-like]
Keep fixed:
- key information:
- chronology:
- required terms or names:

Improve:
- sentence order:
- readability on first pass:
- image or subject landing:
- paragraph rhythm:

Avoid:
- turning explanation into dialogue
- adding new lore
- replacing clear terms with ornate wording

Return mode: [diagnosis + rewrite / rewrite only]
```
