# Continuity Checks

Use this file when contradictions or state drift are the main risk.

## Check These State Layers

- timeline and elapsed time
- location and travel logic
- injuries, fatigue, and recovery
- possession and resource state
- relationship status
- known information and suspicion state
- world rules and exceptions

## High-Risk Questions

- Could this character know this yet
- Could they physically be here yet
- Did the injury, debt, promise, or object vanish
- Did a reveal change later behavior
- Did the rule break only because the plot needed it

## Repair Order

1. fix hard contradiction
2. fix knowledge-state mismatch
3. fix relationship-state mismatch
4. fix soft atmosphere or memory inconsistency

Hard contradictions matter first because they damage trust immediately.
