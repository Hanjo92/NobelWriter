# Diagnostic Workflow

Use this file when the user needs the full order for auditing an existing long-form manuscript or arc run.

## Workflow Order

1. Confirm the audit range.
2. Capture the visible complaint.
3. Form two or three failure hypotheses.
4. Pick the audit mode: triage, checkpoint, or full diagnosis.
5. Check structure before line-level symptoms.
6. Audit pacing and repetition.
7. Check continuity and knowledge state.
8. Audit payoff density and return quality.
9. Add serialization checks if the work is installment-driven.
10. Rank issues by reader impact and repair cost.
11. Deliver a revision brief.

## Start From Reader Symptom

Translate vague complaints into testable forms:

- "boring" -> pacing collapse, low consequence density, repeated scene function
- "messy" -> continuity drift, weak structural spine, unstable focus
- "characters feel off" -> motivation drift, relationship reset, voice flattening
- "ending feels weak" -> broken promise line, insufficient escalation, mistimed payoff

## Audit Narrow Before Broad

If the problem appears in one arc or one volume, audit that range first. Expand only if the root cause clearly predates the local failure.

## Find The First Break Point

Do not stop at the chapter where the reader complaint becomes loud. Try to locate:

- the first compromised setup
- the first repeated beat that should have escalated
- the first continuity miss that later multiplies
- the first motivation slip that makes later scenes feel false

The first break point is usually the best repair entry.

## Rank By Damage

Rank issues with this order in mind:

1. breaks that confuse the reader
2. breaks that reduce forward motion
3. breaks that erase emotional investment
4. breaks that weaken polish but not story function

## Confidence Rule

Mark findings as:

- high confidence: directly evidenced in the text or planning materials
- medium confidence: strongly suggested but still inferential
- low confidence: plausible hypothesis that needs broader reading

Do not present a low-confidence inference as a settled flaw.
