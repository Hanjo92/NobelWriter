# Pacing Checks

Use this file when the story feels slow, circular, or weakly sequenced.

## Diagnose By Scene Function

Mark what each chapter or scene actually does:

- advance conflict
- reveal information
- change relationship state
- impose cost
- reposition the plot
- recap or stall

If many consecutive scenes perform the same function with similar intensity, pacing is probably collapsing.

## Common Pacing Failures

- recap replacing consequence
- cliffhangers that do not create a different next chapter
- repeated hesitation before the same commitment
- exposition between pressure beats
- escalation that is larger in scale but equal in emotional value

## Repair Direction Rule

Report pacing failure as needing changed sequence value, not generic shortening. A short stagnant scene is still stagnant.
