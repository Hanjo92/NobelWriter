# Payoff Checks

Use this file when setup, foreshadowing, reveals, or returns are underperforming.

## Audit Promises

Track:

- what the story promised early
- what was delayed intentionally
- what was forgotten
- what returned too weakly
- what paid off on the wrong emotional track

## Failure Types

### No Return

The setup never matters again.

### Thin Return

The setup returns, but not with enough consequence or reframing value.

### Misaligned Return

The setup promised one kind of emotional or plot result, but paid off in another lane.

### Late But Empty Return

The story remembers the setup, but the reader no longer cares because pressure moved elsewhere.

## Repair Direction Rule

When reporting payoff repair direction, point toward:

1. reconnecting it to current stakes
2. making the return alter relationships or decisions
3. showing how prior material changes meaning now
